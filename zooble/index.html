Zooble is a game prototype created by Mike Kasprzak.  It was developed over the weekend in early January 2004, for the 72 Hour GDC game making competetion.  It was released to the public via Mike's website in Febuary, but never acquired the same support or following as Mike's previous prototype, PuffBOMB.
